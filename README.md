# 1. Descrição

O Readify é um aplicativo de leitura que funciona como uma biblioteca digital pessoal. Nele, o usuário pode registrar os livros que já leu, organizá-los e atribuir avaliações.

# 2. Funcionalidades

Acompanhar leitura atual
Listagem da biblioteca
Avaliação dos livros
Tela de Perfil

O projeto foi desenvolvido com fins educacionais, aplicando conceitos de desenvolvimento de sistemas, como organização de dados, construção de interfaces e interação entre telas.

# 3. Tecnologias
JavaScript
React / React Native
HTML e CSS
